# 🌍 Travel Planner - Complete Web Application

> A modern, responsive travel planning web application built with vanilla HTML, CSS, and JavaScript. Perfect for learning web development!

---

## 🎯 Quick Start

### 1️⃣ Open in VS Code
```bash
File → Open Folder → Select travel-planner
```

### 2️⃣ Install Live Server
```
Ctrl+Shift+X → Search "Live Server" → Install
```

### 3️⃣ Run Application
```
Right-click index.html → Open with Live Server
```

**Done! 🎉 Application opens in your browser**
